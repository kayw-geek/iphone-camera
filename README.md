# iphone-camera
:iphone: 模拟iPhone 实现了拍照、手电筒、上滑底部横条等功能

Features

目前功能

- 实时时间日期
- 手电筒功能
- 拍照功能
- 拍照图库预览图功能
- iPhone底部横条上拖返回功能

Todolist

- 更换手机外壳颜色
- 更换壁纸
- 相机滤镜等功能实现
- 进入桌面
- 模拟Safari
- 图库
- 音乐
